Title Principal
CEGED

El CEGED (centro de estudios para la gestión del desarrollo) plantea como uno de sus objetivos promover y ejecutar la investigación y capacitación con vista al desarrollo sostenible, por lo que necesita de un sitio web que permita la muestra y promoción de las labores llevadas a cabo de este centro. El objetivo de este proyecto consiste en desarrollar un sitio web que permita mostrar la información científico-académica del CEGED..

Tarea

El CEGED (Centro para la Gestión del Desarrollo), antes conocido como CEDAR (Centro para la Gestión del Desarrollo Agrario y Rural) se ha dado a la tarea de lograr el desarrollo sostenible. Para lograr esto, este imparte capacitaciones (cursos, seminarios, etc.â€¦) para el uso y manejo del conocimiento para crear las herramientas necesarias para la gestión económica tecnológica y medioambiental sostenible.

Misión

CEGED tiene como misión promover y ejecutar la investigación, capacitación y superación en el desarrollo, uso y manejo de sistema de conocimientos y de información agrario y rural con el objetivo de desarrollar programas y actividades de investigación, asesoramiento, capacitación, estudio y diseminación, tendientes al avance del conocimiento y de la información agraria y rural con para así promover procesos de desarrollo local sustentables.
